# 功能设计：登录

## 1. 功能描述
用户通过输入员工番号和密码进行系统登录，支持密码变更、错误提示、退出功能。

## 2. 功能实现流程
1. 用户输入员工番号和密码
2. 点击"登录"按钮，系统校验账号密码
3. 密码错误时，显示提示信息
4. 登录成功后进入主页面
5. 支持"密码变更"跳转、"结束"关闭系统

## 3. 业务规则
- 员工番号和密码均为必填项
- 密码错误时，提示"密码不正确"
- 账号状态为禁用时，禁止登录
- 支持密码变更流程

## 4. 使用角色
- 所有系统用户

## 5. 界面设计要求
- 两个输入框：员工番号、密码
- 两个按钮：登录、结束
- 一个"密码变更"链接
- 一个隐藏的错误提示框
