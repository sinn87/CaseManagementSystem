# 功能设计：案件一览

## 1. 功能描述
用户可通过多条件组合查询案件，结果以表格形式展示，支持双击查看详细。

## 2. 功能实现流程
1. 用户点击"案件一览"进入页面
2. 顶部图片按钮返回主页面，右上角"返回"按钮返回上一页
3. 上部11个搜索框，支持多条件组合查询
4. 点击"搜索"按钮，组合SQL筛选DataGridView
5. 下部DataGridView展示案件一览，双击行进入详细页面

## 3. 业务规则
- 支持多条件组合查询
- 搜索条件可为空
- 双击行进入详细页面

## 4. 使用角色
- 所有系统用户

## 5. 界面设计要求
- 11个搜索框，DataGridView表格
